</br> ## 简介 + 湖北省孝感高级中学位于湖北省孝感市，解放初期就是省级重点中学，是湖北省示范学校、湖北省教育科学研究实验中学。 ## 历史沿革 + 湖北省孝感高中成立于1946年，原名为湖北省立孝感中学，位于孝感市交通西路。 + 解放初期，更名为省立孝感高中，是全省18所省级重点中学之一。后更名为湖北省孝感高级中学。 “文革”期间停办。 + 1971年，学校恢复重建。 + 1998年，被湖北省政府教育督导室评为“省级示范学校”。 + 2000年，学校安排一个考察小组赴杭州四中学习后，开始借鉴并运用“学生值周管理办法”。 + 2001年，学校实施异地建设。 + 2002年9月，整体搬迁至交通西路，占地300亩，建筑面积10万余平方米。 + 2003年后，高中扩张，一个年级最高曾扩招到32个班。 + 2003年5月1日看，正式成立孝感高中中澳国际合作部。 + 2004年，经省复评保留省级示范学校称号。 + 2010年，“省级示范高中”复评合格。 + 2013年，湖北省要求清理“一校两制”和“校中校”问题，学校学生人数下降。 ## 文化传统 ### 校徽 > 校徽主体图案以孝感高级中学的简称“孝高”拼音的第一个字母“XG”为设计元素，经过艺术加工后变形为“龙”和“帆”的形状。寓意学校所确立的“兴学立德，培源则栋”的办学理念。 > > 校徽以蓝色为主色调，既象征无数学子尽情遨游在知识的海洋，又暗 含校园蓝瓦白墙的建筑色彩风格。 ### 校训 + 有为有守，不忮不求 > “有为”，指孝高人在学习、工作、事业诸方面都要有所作为；“有守”，指孝高人在日常生活及一言一行中都要有良好操守。“有为”，是鼓励孝高人在人生观和价值观上时刻要积极进取；“有守”，是培养孝高人在道德品质和法制观念上要严于律己。“有为兼有守”，就是要把孝高人锻炼成志向远大、操行端正的优秀人才。 > > “不忮不求”，语出《诗经·风·雄雉》。“不忮”，指孝高人在人际交往中不要忌恨人，不要害人；“不求”，指孝高人在社会活动中不要贪心，不要一味索取。“不忮”，是要求孝高人在人与人的关系中保持宽容大度的胸怀；“不求”，是要求孝高人在钱财、待遇等方面保持廉洁清明的本色。“不忮又不求”，就是要把孝高人磨练成心胸开阔、行事高洁的现代公民。 ### 校风 + 笃行慎思，敏学善育 > 笃行慎思：出自《中庸》的一句话：“博学之，审问之，慎思之，明辨之，笃行之”。意思是必须广博地学习，审慎地询问，慎重地思索，明晰地辨析，踏实地履行，才能真正达到理想的学问境界和人生境界。 ### 学校精神 + 崇尚卓越，追求一流，拒绝平庸的拼搏精神
